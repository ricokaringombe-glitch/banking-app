# Banking App API

A RESTful banking application built with Spring Boot and PostgreSQL.

## Technologies Used
- Java 25
- Spring Boot 4.0
- PostgreSQL 18
- Spring Data JPA
- Spring Security
- Lombok
- Maven

## Getting Started

### Prerequisites
- Java 25
- PostgreSQL
- Maven

### Setup
1. Clone the repository
2. Create a PostgreSQL database called `bankingdb`
3. Update `application.properties` with your database credentials
